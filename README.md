# projekt_ki

## Aufgabe
- Präsentationsplan erstellen für die Projektwoche

Allgemein:
- 5 Tage (Mo, Di, Mi, Do, Fr)
- 4 Slots:
    - 1: 08:00 - 09:00
    - 2: 10:00 - 11:00
    - 3: 13:00 - 14:00
    - 4: 15:00 - 16:00

Studiengang:
- Anzahl an Projektgruppen
- 1-3 Prüfungskomissionen

Gruppen:
- 3 Präsis halten
- Tages-Slot-Raum-Termin
- keine parallenen Termine

Laborraum:
- 3 (L1, L2, L3)
- 1 Präse in jedem Zeitslot

extra:
- Wenn Gruppe Solt 4 hat, darf diese nicht 1. Slot haben am direkt nächsten Tag
- Prüfungskomission darf an einem Tag max 1x den Raum wechseln
- Begründen, falls nicht möglich
- weiches Constraint:
Prüfungskommissionen sollten an einem Tag nicht länger als nötig im Haus gehalten werden,
d.h. wenn ein früherer Präsentationsslot möglich ist, so ist dieser zu nutzen.

## Daten
TxT-Datei aufgebaut wie eine CSV mit ";"-Seperator

`Studiengang;Projektgruppe;Kommissionen`
